# 🎨 HTML/CSS Exercises for Beginners

A collection of **37 hands-on exercises** built with HTML and CSS to practice real-world layouts and styling techniques.

🔗 **Live Preview:** [sdvp-3.github.io/01-projectp](https://sdvp-3.github.io/01-projectp/)

---

## 🛠️ Technologies Used

- **HTML** — Structure and content of every page
- **CSS** — Styling and visual design
- **Flexbox** — Flexible, responsive layouts made easy
- **CSS Grid** — Perfect for structuring layouts into rows and columns
- **Position** — Mastering `absolute`, `relative`, and `fixed` placement
- **Fonts** — Practicing Arial, Tahoma, Verdana and more
